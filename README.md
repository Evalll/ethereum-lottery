# ethereum-lottery

基于智能合约的彩票系统

## 环境要求

1. web3
2. MetaMask
3. RinkeBy 测试网络
4. npm

## 开发过程

1. 在 RemixIDE 上部署合约；
2. 利用 npm 的 create-react-app 快速建立好一个 ReactJS 的范本，利用 Web3 来与进行交互。

## 部署

```
$ git clone https://github.com/Evalll/ethereum-lottery.git
$ cd ethereum-lottery
$ cd lottery-react
$ npm start
```
